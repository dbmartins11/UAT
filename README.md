# UAT

## Get started
1. Install dependencies
   ```bash
   npm install
   ```
2. Start the app
   ```bash
    cd Ubiqutous
    npx expo start
   ```

# Project Structure

1. Page are in folder ***app/(tabs)*** (ex: index.tsx, shop.tsx, ...)
2. Components are in folder ***components***.
3. Images and icons should be in ***assets***.
